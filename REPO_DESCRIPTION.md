# REPO_DESCRIPTION

**Title**: X to Kindle (Terminal Edition)

**Description**: A minimalist, stateless web utility that converts long-form X (Twitter) Articles into EPUB files and delivers them directly to your Kindle. Built with Next.js, Puppeteer, Resend, and pure Geist Pixel aesthetics. 

**Tags**: `nextjs`, `kindle`, `epub`, `web-scraping`, `minimalist`, `utility`
